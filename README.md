# Homebrew Tap for HomeRun

[HomeRun](https://github.com/aGallea/homerun) — manage GitHub Actions self-hosted runners on macOS.

## Install

```sh
brew tap aGallea/homerun

# CLI tools (homerun + homerund)
brew install homerun

# Desktop app
brew install --cask homerun
```

## Upgrade

```sh
brew upgrade homerun
brew upgrade --cask homerun
```
